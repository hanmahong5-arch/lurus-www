# Story 2.2: Navbar 基础结构与 Logo

Status: done

## Story

As a 访客,
I want 看到固定在顶部的导航栏和品牌 Logo,
so that 我可以随时访问网站的主要区段，并识别这是 Lurus 品牌。

## Acceptance Criteria

1. **AC1**: 导航栏固定在视口顶部 (position: fixed)
2. **AC2**: 左侧显示 Lurus Logo (Caveat 字体)
3. **AC3**: 导航栏初始状态为透明背景
4. **AC4**: 滚动距离 > 0 时背景变为 `bg-cream-50/90 backdrop-blur-sm shadow-sm`
5. **AC5**: 使用语义化 `<header>` 和 `<nav>` 标签
6. **AC6**: Logo 是可点击的链接，指向首页
